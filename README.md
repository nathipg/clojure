# clojure

IntelliJ Shortcuts

Note: Check some shorcuts in Edit -> Structural Editing

- Alt + 1 => Hide/Show project panel
- Alt + Shift + L => Load file in REPL
- Alt + Shift + J => Remove form from parenthesis
- Alt + Shift + K => Add next form to parenthesis
- Alt + Shift + P => Run current form
- Ctrl + Alt + L => Reformat code
